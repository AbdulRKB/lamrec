## 📋 lamrec  
Your simple and fast record-keeping tool.

## 🚀 Installation & Usage
```bash
$ git clone https://github.com/AbdulRKB/lamrec.git
$ cd lamrec
$ pip install -r requirements.txt
$ python app.py
```
